# Markdown利器

工具：

- PicGo
- Typora
- 图床服务

## PicGo

图床服务，支持列表：

- 腾讯云COS
- 阿里云OSS
- SM.MS
- GitHub
- 七牛云
- Imgur
- 又拍云

## Typora

在 `文件 ` --> `偏好设置`

![Typora Picgo设置](https://s2.loli.net/2023/03/09/iDQtS4ymeljNEF1.png)

这样在每次插入本地图片的时候，就可以支持把图片上传到指定的图床服务上。大大提高了写文档的效率。
